# Сервис по бронированию отелей

## Описание

Сервис позволяет:
- Авторизация и регстрация пользователей
- Бронировать, снимать бронь
- Создавать записи по отелям
- Просматривать список доступных отелей

## Технологии

- FastAPI
- SQLAlchemy
- Alembic
- Redis
- Celery
- PyJWT
- PostgreSQL
